# SchoolSaavy Frontend

A professional, corporate-styled React frontend for the SchoolSaavy School Management System.

## Live Backend
- **API:** https://schoolsaavy-backend-updated.onrender.com/api
- **Health Check:** https://schoolsaavy-backend-updated.onrender.com/api/health

## Login Credentials
- **Email:** superadmin@schoolsaavy.com
- **Password:** SuperAdmin@123
- **User Type:** Super Admin

## Features
- Login with role selection (Super Admin, School Admin, Teacher, Parent)
- Dashboard with stats, quick actions, recent activity
- Student Management
- Teacher Management
- Attendance Tracking
- Fee Management
- Exams & Results
- API Health Monitor
- Fully responsive design

## Tech Stack
- React 18
- Outfit + Fraunces fonts
- Navy (#1e3a5f) + Gold (#c9963b) theme
- No external UI library — pure CSS

## Setup
```bash
npm install
npm start
```

## Deploy
Push to GitHub, then connect to Vercel for instant deployment.
